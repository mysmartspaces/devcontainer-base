## Alpine Available Setups

- Alpine SDK (alpine-sdk)
- Dapr CLI (dapr-cli)
- Ghostscript (ghostscript)
- Go (go, go-tools)
- ImageMagick (imagemagick)
- Kubernetes
  - kubernetes
  - helm
  - kustomize
  - minikube
  - kind
  - skaffold
  - tilt
- Node.js (node)
- Packer (packer)
- Python 2 (python2)
- Python 3 (python3)
